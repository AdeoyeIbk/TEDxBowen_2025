This is the official website for TEDxBowenUniversity 2025.

## Setup

Prerequisites

- Node.js (recommended 18 or newer)
- npm (or yarn / pnpm)

Install

1. Install dependencies:

```bash
npm install
```

Run locally

```bash
npm run dev
# open http://localhost:5173
```

Build & preview

```bash
npm run build
npm run preview
```
